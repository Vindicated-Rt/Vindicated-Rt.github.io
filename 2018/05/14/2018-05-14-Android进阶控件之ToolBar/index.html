ToolBar是什么

打开Android 开发者文档:

public class Toolbar extends ViewGroup

java.lang.Object

    ↳ android.view.View

       ↳ android.view.ViewGroup

            ↳ android.support.v7.widget.Toolbar

ToolBar是android.support.v7.widget中的一个类,继承自ViewGroup,它用于程序内容中的标准工具栏。
